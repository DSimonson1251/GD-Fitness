# GD-Fitness
GW+DS app

## Local setup (Python / FastAPI)

1. Create a Python virtual environment and activate it:

	python3 -m venv .venv
	source .venv/bin/activate

2. Install dependencies:

	pip install -r requirements.txt

3. Run the app:

	./start.sh

4. Open http://localhost:8000/health to verify the server is running.

If you want a different stack (Node, mobile, etc.), tell me and I will scaffold that instead.
